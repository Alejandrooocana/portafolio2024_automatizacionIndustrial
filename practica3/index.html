- Resumen -

Para realizar esta práctica de comunicación asíncrona, se utilizó la conversión de datos de decimal a binario mandando cada uno de los bits, junto con un pulso inicial, a través de un pin a otro en los arduinos, separados por cierta cantidad de tiempo lo que nos permite que el segundo arduino obtenga esta información para volver a realizar la conversión de binario a decimal.

p>

- Introducción -

Hoy en día podemos notar que existe la transmisión de millones y millones de datos mediante redes de interconexión los cuales requieren ciertos protocolos de comunicación para poder lograr que se haga de manera efectiva, según sean las necesidades y conexiones que se estén utilizando.

Estos protocolos de comunicación son un conjunto de pautas o instrucciones que funcionan como guía para el intercambio de información, por lo que son necesarias para que un programa o una máquina las ejecute de manera correcta y organizada.

Estas pautas que regulan la comunicación incluyen instrucciones para posibilitar tanto la identificación, como la conexión de dispositivos entre sí, además de establecer normas de formato para asegurar un flujo seguro de información desde el emisor hasta el receptor. En caso de errores durante la transmisión de datos, los protocolos de red se encargan de detectar y corregirlos de manera inmediata. Este enfoque beneficia a las compañías al reducir costos en tecnologías de la información y mejorar la eficiencia operativa.

Al ser una herramienta muy importante para nuestra vida diaria, a continuación, se realizará una práctica con el fin de aprender sobre estos protocolos de manera más visual utilizando un contador de pulsos y uno de tiempo.
